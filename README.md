### Install
Clone this project

```bash
> git clone https://github.com/kizzajamie/wabot
> cd wabot
```

Install the dependencies:

```bash
> npm i
> npm i -g sharp
> npm i -g pm2
```

### Usage
**1. run the Whatsapp bot**

```bash
> npm start
or 
> node .
```
or
```bash
> pm2 start .
> pm2 monit
```
**2. stop the Whatsapp bot**

for npm :
```bash
> ctrl + c
```
for pm2 :
```bash
> pm2 stop
```
